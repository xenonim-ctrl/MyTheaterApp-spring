О разработчике

Информация о проекте

Информационная система для управления театром: спектакли, показы, статистика и пользователи. Веб-приложение реализовано на Java + Spring Boot с использованием трехслойной архитектуры и PostgreSQL.

Технологический стек

  • Backend: Java 17, Spring Boot 3.5.0
  • Framework: Spring MVC, Spring Data JPA, Spring Security
  • База данных: PostgreSQL
  • ORM: Hibernate (JPA)
  • Frontend: Thymeleaf, HTML5, CSS3, JavaScript
  • UI Framework: Bootstrap 5.3.0
  • Графики: Chart.js
  • Build Tool: Maven

Архитектура

Приложение построено на основе трехслойной архитектуры:

  • Presentation Layer: Thymeleaf шаблоны, REST API контроллеры
  • Business Layer: Сервисы (PlayService, PerformanceService, StatisticsService)
  • Data Access Layer: JPA репозитории

Основные возможности

  • Регистрация и аутентификация пользователей
  • Ролевая модель доступа (OWNER/ADMIN/USER)
  • CRUD операции для спектаклей и показов
  • Многоуровневая модель данных (Спектакль → Показ)
  • Фильтрация по множественным критериям
  • Сортировка по различным полям
  • Пагинация результатов
  • Статистика с графиками и расчетами
  • RESTful API
  • Защита от CSRF атак
  • Глобальная обработка исключений

Безопасность

Приложение использует Spring Security для обеспечения безопасности:

  • Хеширование паролей (BCrypt)
  • CSRF защита
  • Защита от SQL инъекций (через JPA)
  • Ролевая авторизация
